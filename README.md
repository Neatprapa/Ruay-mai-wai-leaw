# Green Bistro Manager 🌿
A C++ restaurant management game focused on sustainability and SDG goals.

## Features
- **Carbon Tracking**: Every ingredient adds to your carbon footprint.
- **BST System**: Records daily performance using a Binary Search Tree for sorting.
- **SDG Missions**: Complete 3 random missions out of 5 to achieve higher ranks.
- **Ranking**: S, A, B, and F ranks based on carbon points and mission success.
